Site navigation

The openTECR site is designed to seamlessly allow researchers to query and download experimental thermodynamics data of enzyme-catalyzed biochemical reactions. The query entries in the Query page are simply populated with the desired search criteria, and then the search results are depicted in the space below the query criteria.
